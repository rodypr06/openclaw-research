# Quick Start Guide

## Launch the Development Server

```bash
cd /Users/rrabelo/.openclaw/workspace/openclaw-research
npm run dev
```

The website will open automatically at http://localhost:3000

## What You'll See

1. **Hero** - Animated landing with code preview
2. **Stats Dashboard** - Quick metrics at top
3. **Sentiment Analysis** - Overall sentiment gauge & trends
4. **Timeline** - Release roadmap with milestones
5. **Community** - Interactive charts & analytics
6. **Tools** - 12+ tools with detailed info
7. **Skills** - Capabilities directory with proficiency levels
8. **Visualizations** - Multiple chart types
9. **News** - Featured articles & trending topics

## Key Interactions

- **Hover** over cards for subtle scale effects
- **Click** on timeline events to see details
- **Click** on tools/skills for expanded information
- **Use tabs** to switch between chart views
- **Filter** news by category
- **Mobile menu** - Click hamburger icon

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide Icons

## File Locations

- Main app: `src/App.jsx`
- Components: `src/components/`
- Styles: `src/index.css`
- Config: `tailwind.config.js`

## Build for Production

```bash
npm run build
```

Output: `dist/` directory
